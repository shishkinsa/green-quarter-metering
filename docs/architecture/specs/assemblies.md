# Именование сборок

Сборки именуются по шаблону: `<Prefix>.<НазваниеСервиса>.<Назначение>`.

Примеры для шаблона: `GQ.WebApi.Entities`, `GQ.WebApi.DataAccess.Postgres` (префикс `SP` заменяется на префикс вашего проекта).

Слои backend и frontend — в отдельных спеках:

- [backend.md](backend.md)
- [frontend.md](frontend.md)
