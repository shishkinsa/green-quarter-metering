# Цели проекта

Система учёта электроэнергии для жилого комплекса **«Зелёный квартал»**.

## Бизнес-ценность

- **Проблема:** централизованный учёт потребления электроэнергии в ЖК, контроль показаний и справочников.
- **Пользователи:** энергетик Василий (ответственный за систему), управляющая компания, жильцы (в перспективе).
- **Метрики успеха:** актуальность данных учёта, сокращение ручного ввода, прозрачность потребления по объектам ЖК.
- **Заинтересованные стороны:** энергетическая служба ЖК, бухгалтерия, жильцы.
