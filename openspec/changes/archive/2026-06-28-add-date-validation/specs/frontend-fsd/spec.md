## MODIFIED Requirements

### Requirement: Страница квартир дома

Страница квартир SHALL отображать квартиры выбранного дома в таблице с данными владельца, датой поверки счётчика и сводкой по показаниям.

#### Scenario: Отображение таблицы квартир

- **WHEN** пользователь открывает страницу квартир дома
- **THEN** отображается заголовок с наименованием дома и кнопка «Добавить квартиру»
- **AND** таблица содержит колонки: **id**, **номер квартиры**, **владелец**, **дата поверки счётчика**, **последняя дата передачи показаний**, **показание**, **статус передачи**
- **AND** колонка «дата поверки счётчика» показывает значение в формате `ДД.ММ.ГГГГ` или прочерк, если дата не задана

#### Scenario: Добавление квартиры через модальную форму

- **WHEN** пользователь нажимает «Добавить квартиру»
- **THEN** открывается модальное окно с полями номер квартиры, этаж (опционально) и **дата поверки счётчика** (date picker)
- **AND** date picker не позволяет выбрать дату позже текущего дня
- **WHEN** пользователь сохраняет форму
- **THEN** квартира создаётся через API с указанной датой поверки (или `null`, если поле пустое)
- **AND** таблица квартир обновляется

#### Scenario: Редактирование квартиры через модальную форму

- **WHEN** пользователь выбирает действие «Редактировать» в строке квартиры
- **THEN** открывается модальное окно с предзаполненными полями номер, этаж и **дата поверки счётчика**
- **AND** date picker не позволяет выбрать дату позже текущего дня
- **WHEN** пользователь сохраняет изменения
- **THEN** квартира обновляется через API
- **AND** колонки «номер квартиры» и «дата поверки счётчика» обновляются

#### Scenario: Редактирование владельца квартиры

- **WHEN** пользователь выбирает действие редактирования владельца (кнопка в строке или отдельное действие)
- **THEN** открывается модальное окно с полями ФИО и телефон
- **WHEN** пользователь сохраняет
- **THEN** владелец сохраняется через API
- **AND** колонка «владелец» обновляется

#### Scenario: Переход к истории показаний квартиры

- **WHEN** пользователь кликает по строке таблицы квартир (не по кнопкам действий)
- **THEN** выполняется переход на страницу показаний квартиры (`/apartments/{apartmentId}/meter-readings`)

#### Scenario: Удаление квартиры

- **WHEN** пользователь нажимает «Удалить» в строке квартиры
- **THEN** отображается подтверждение удаления
- **WHEN** пользователь подтверждает
- **THEN** квартира удаляется через API
- **AND** строка исчезает из таблицы
